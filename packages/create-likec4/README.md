## Scaffolding Your First LikeC4 Project

> **Compatibility Note:**
> LikeC4 requires [Node.js](https://nodejs.org/en/) version 18+, 20+

With NPM:

```bash
$ npm create likec4@latest
```

With Yarn:

```bash
$ yarn create likec4
```

With PNPM:

```bash
$ pnpm create likec4
```

Then follow the prompts!
