# Customer: Mobile application

We can define a view to explain interactions with mobile application.

```likec4
views {

  view mobileApp of mobileApp {
    include
      *,
      internetBankingSystem,
      internetBankingSystem.apiApplication,
      mobileApp -> internetBankingSystem.apiApplication.*

    style * {
      color muted
    }

    style apiApplication.*, mobileApp {
      color primary
    }

    style customer {
      color secondary
    }
  }

}
```
