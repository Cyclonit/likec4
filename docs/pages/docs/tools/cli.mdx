# CLI

The `likec4` CLI is a tool for managing and validating your LikeC4 projects and generating the diagrams

## Install

<Tabs items={['npm', 'yarn']}>
  <Tab>
    ```bash
    npm install --save-dev @likec4/cli
    ```
  </Tab>
  <Tab>
    ```bash
    yarn add -D @likec4/cli
    ```
  </Tab>
</Tabs>
