# Specification

In `specification` you customize your model.

## Element

`element` defines the kind that can be used in the model.

```likec4 copy
specification {

  // Whatever you want
  element user
  element cloud
  element system
  element application
  element component
  element microservice
  element queue
  element graphqlMutation
  element repository
  element database

}
```

Definition may have a nested `style` - defaults for all elements of this kind.

```likec4 copy
specification {

  element user {
    style {
      shape person
    }
  }

  element singlePageApp {
    style {
      shape browser
      color amber
    }
  }

  element queue {
    style {
      shape queue
      color gray
    }
  }  

}
```

### Shapes

The following element shapes are available:

| shape        |  |
| :------------ | :---------: |
| rectangle (default)       |
| storage, cylinder |  |
| browser |   |
| person     |  |
| queue |   |
