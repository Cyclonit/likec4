# Specification

In `specification` you customize your model.

## Element kind

`element` defines the kind that can be used in the model.

```likec4 copy
specification {

  // Define whatever you want
  element user
  element cloud
  element system
  element application
  element component
  element controller
  element microservice
  element queue
  element restapi
  element graphqlmutation
  element repository
  element database
  element pgtable

}
```

Definition may have a nested `style` - defaults for all elements of this kind.

```likec4 copy
specification {

  element user {
    style {
      shape person
    }
  }

  element spa {
    style {
      shape browser
      color amber
    }
  }

  element queue {
    style {
      shape queue
      color gray
    }
  }

}
```

### Shapes

The following element shapes are available:

| shape     | example |
| :-------- | :-----: |
| rectangle |   ..    |
| storage   |   ..    |
| cylinder  |   ..    |
| browser   |   ..    |
| mobile    |   ..    |
| person    |   ..    |
| queue     |   ..    |

By default, the shape is `rectangle`.

### Colors

| color          |                                                           |
| :------------- | :-------------------------------------------------------: |
| primary, blue  |  <div className="bg-blue-500 rounded-sm w-8 h-6"></div>   |
| secondary, sky |   <div className="bg-sky-600 rounded-sm w-8 h-6"></div>   |
| muted, slate   |  <div className="bg-slate-600 rounded-sm w-8 h-6"></div>  |
| amber          |  <div className="bg-amber-600 rounded-sm w-8 h-6"></div>  |
| gray           | <div className="bg-neutral-500 rounded-sm w-8 h-6"></div> |
| green          |  <div className="bg-green-600 rounded-sm w-8 h-6"></div>  |
| indigo         | <div className="bg-indigo-600 rounded-sm w-8 h-6"></div>  |
| red            |   <div className="bg-red-500 rounded-sm w-8 h-6"></div>   |

By default, the color is `primary`.

## Tag

Tags are used to mark elements in the model, like `#deprecated`, `#epic123` or `#team2`.

```likec4
specification {

  tag deprecated
  tag epic123
  tag team2

}
```
