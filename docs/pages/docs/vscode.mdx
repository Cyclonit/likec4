LikeC4 has an official [extension for VSCode](https://marketplace.visualstudio.com/items?itemName=likec4.likec4-vscode).

The extension provides (via [LSP](https://microsoft.github.io/language-server-protocol)):

- Validation and error reporting
- Previews
- Code completion and navigation 
- Resolve references (like `find all references`, `go to definition` .. )
- "Safe" renames
- Syntax highlighting
- Hover information

Extension is universal and can run as web-version.  
You can open [this sample](https://github.dev/likec4/example-cloud-system) in the browser (install workspace recommendation and refresh the page).
